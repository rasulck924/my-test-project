# Өнімділік пен мониторинг құралдары

- JMeter – нагрузкалық тест жүргізу құралы  
- Locust – Python арқылы жүктеме тесті  
- k6 – JavaScript негізіндегі жүктеме тесті  
- Gatling – жоғары өнімді нагрузкалық тест  
- Grafana – метрикаларды визуализациялау  
- Prometheus – метрикаларды жинау және сақтау  
- InfluxDB – уақыттық метрикалар дерекқоры  
- ELK Stack (Elasticsearch, Logstash, Kibana) – логтарды талдау  
- New Relic / Datadog – қосымша өнімділігін бақылау (APM)
